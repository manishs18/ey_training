If by interchat memory you mean "remember things across different conversations" and intrachat memory means "remember things within the current conversation," then ChatGPT handles them differently:

Memory Type	What it does	How to configure
Intrachat memory (current chat context)	Remembers everything that fits within the context window of the current conversation.	No user configuration. It happens automatically within a chat.
Interchat memory (cross-chat memory)	Lets ChatGPT use information from previous conversations and saved memories in future chats.	Configure via Memory settings.
How to enable cross-chat (interchat) memory
Open Settings → Personalization.
Turn on:
Reference saved memories
Reference chat history (optional but recommended)

Difference between the two settings:

Reference saved memories = persistent facts explicitly remembered (e.g., "I'm a Python developer"). These remain until deleted.
Reference chat history = ChatGPT can mine past conversations for useful context and preferences. These memories are dynamic and may change over time.
How to inspect what ChatGPT remembers

Go to:

Settings → Personalization → Manage Memories

There you can:

View memories
Delete individual memories
Delete all memories
Refresh the memory summary (newer memory system)
For project-specific interchat memory

If your goal is something like:

"Remember all discussions about my Appium automation project, but keep them separate from my personal chats"

The most effective approach is to use Projects and keep related conversations inside the same project. Recent memory improvements also allow ChatGPT to reference prior conversations and surface memory sources more explicitly.
